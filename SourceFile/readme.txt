lynk03-nfs-archived.rar 是Blender工程，已经导出obj和mtl,可以用NFSMW12VMT车辆模型工具直接导入。

lynk03-nfs-archived.rar is packaged Blender project, has been exported obj and mtl,can using NFSMW12VMT Vehicle Model Tool to import.

使用Noesis解包/打包BNDL文件。

use Noesis plugin to unpack/repack BNDL file.

使用NFSbndlModelChallenger调节驾驶员和轮子的位置。

use NFSbndlModelChallenger adjust location of the driver and wheel.


但是，当解包VEH_122701_LO.BNDL然后用NFSMW12VMT导入，使用Noesis plugin打包的VEH_122701_LO.BNDL会导致游戏崩溃，原因未知。

But,When unpack VEH_122701_LO.BNDL then use NFSMW12VMT import,use Noesis plugin repacked VEH_122701_LO.BNDL will make game crash, reason is unknown.
